# AI集成模块界面设计

## 1. 模块概述

AI集成模块提供智能图像生成、文案创作和智能建议功能，帮助设计师提升创作效率。界面设计强调创作流程的直观性和AI生成结果的可视化展示，采用工作室风格的布局。

## 2. AI功能入口

### 2.1 AI工具面板

**面板位置**: 右侧固定面板或底部浮动工具栏

**工具分类**:
- **图像生成**: 场景生成、风格转换、材质替换、光照调整
- **文案创作**: 提案文案、设计理念、产品描述、多语言翻译
- **智能建议**: 配色建议、布局优化、预算分配

**面板样式**:
- **背景色**: `#FFFFFF`
- **阴影**: `0 4px 16px rgba(0, 0, 0, 0.12)`
- **圆角**: `8px`
- **内边距**: `16px`

### 2.2 快速访问入口

**全局快捷按钮**:
- **位置**: 右下角固定位置
- **样式**: 圆形按钮，直径 `56px`，背景色 `#D4AF37`（金色）
- **图标**: AI魔法棒图标，白色
- **悬停效果**: 放大至 `60px`，显示工具提示"AI助手"
- **点击展开**: 展开显示AI工具菜单

**工具菜单**:
- **图像生成**: 图标 + 文字
- **文案创作**: 图标 + 文字
- **智能建议**: 图标 + 文字
- **历史记录**: 图标 + 文字
- **设置**: 图标 + 文字

## 3. AI图像生成

### 3.1 场景生成

**界面布局**:
- **左侧输入区**: 宽度 `400px`，包含输入表单
- **右侧预览区**: 宽度自适应，显示生成结果

**输入表单**:
- **场景描述**: 必填，文本域，占位符"描述您想要的室内场景，例如：现代简约风格的客厅，大落地窗，米白色沙发..."
- **空间类型**: 下拉选择器（客厅/卧室/厨房/卫生间/办公室/餐厅/其他）
- **设计风格**: 多选下拉（现代简约/新中式/欧式古典/北欧/工业风/日式）
- **色调**: 颜色选择器或预设色板（暖色调/冷色调/中性色调）
- **光照**: 下拉选择器（自然光/暖光/冷光/混合光）
- **视角**: 下拉选择器（正视图/俯视图/透视图）
- **图片尺寸**: 单选按钮组（1024×1024/1920×1080/自定义）
- **生成数量**: 数字输入框（1-4张）
- **高级选项**: 折叠面板
  - 细节程度：滑块（简洁-丰富）
  - 创意程度：滑块（写实-创意）
  - 参考图片：图片上传器

**生成按钮**: 主要按钮，文字"生成场景"，图标魔法棒

**预览区域**:
- **生成进度**: 显示进度条和预计时间
- **生成过程**: 显示模糊到清晰的渐进式加载
- **结果展示**: 网格布局显示生成的多张图片
- **图片操作**:
  - 放大查看：点击图片全屏查看
  - 下载：下载按钮
  - 应用到提案：添加到当前提案
  - 重新生成：基于此图重新生成
  - 编辑：进入图片编辑器

**生成历史**:
- **历史记录**: 底部显示最近生成的图片缩略图
- **点击查看**: 点击缩略图查看历史生成结果
- **管理**: 删除、收藏、导出

- **左右分屏**: 左侧原图，右侧转换后
- **滑动对比**: 中间滑块，拖动对比
