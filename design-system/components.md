# 组件设计文档

## 1. 按钮组件（Button）

### 1.1 按钮类型

#### 主要按钮（Primary Button）
用于页面中最重要的操作，每个页面区域建议只有一个主要按钮。

**视觉规范**:
- **默认状态**: 背景色 `#D4AF37`（金色），文字色 `#FFFFFF`，圆角 `4px`，内边距 `12px 24px`
- **悬停状态**: 背景色 `#C19B2F`，添加阴影 `0 2px 8px rgba(212, 175, 55, 0.3)`
- **激活状态**: 背景色 `#B08A28`，阴影加深 `0 1px 4px rgba(212, 175, 55, 0.4)`
- **禁用状态**: 背景色 `#E0E0E0`，文字色 `#9E9E9E`，光标 `not-allowed`
- **加载状态**: 显示旋转图标，文字变为"加载中..."，禁用点击

#### 次要按钮（Secondary Button）
用于次要操作或与主要按钮配合使用。

**视觉规范**:
- **默认状态**: 背景色透明，边框 `1px solid #2D2D2D`，文字色 `#2D2D2D`，圆角 `4px`，内边距 `12px 24px`
- **悬停状态**: 背景色 `#F8F5F0`，边框色 `#D4AF37`，文字色 `#D4AF37`
- **激活状态**: 背景色 `#E0E0E0`，边框色 `#C19B2F`
- **禁用状态**: 边框色 `#E0E0E0`，文字色 `#9E9E9E`

#### 文本按钮（Text Button）
用于不需要强调的操作，如取消、返回等。

**视觉规范**:
- **默认状态**: 无背景，无边框，文字色 `#2D2D2D`，内边距 `8px 16px`
- **悬停状态**: 文字色 `#D4AF37`，背景色 `rgba(212, 175, 55, 0.08)`
- **激活状态**: 文字色 `#C19B2F`，背景色 `rgba(212, 175, 55, 0.15)`
- **禁用状态**: 文字色 `#9E9E9E`

#### 图标按钮（Icon Button）
用于工具栏、操作栏等需要紧凑布局的场景。

**视觉规范**:
- **默认状态**: 正方形 `40px × 40px`，圆角 `4px`，图标尺寸 `20px`，图标色 `#2D2D2D`
- **悬停状态**: 背景色 `#F8F5F0`，图标色 `#D4AF37`
- **激活状态**: 背景色 `#E0E0E0`
- **禁用状态**: 图标色 `#9E9E9E`

### 1.2 按钮尺寸
- **大号（Large）**: 高度 `48px`，字体 `16px`，内边距 `14px 32px`
- **中号（Medium）**: 高度 `40px`，字体 `14px`，内边距 `12px 24px`（默认）
- **小号（Small）**: 高度 `32px`，字体 `12px`，内边距 `8px 16px`

### 1.3 按钮组合
- **按钮组**: 多个按钮水平排列，间距 `12px`
- **按钮组优先级**: 主要按钮在右侧，次要按钮在左侧
- **移动端适配**: 按钮组在移动端垂直堆叠，全宽显示

## 2. 输入框组件（Input）

### 2.1 文本输入框

**视觉规范**:
- **默认状态**: 高度 `40px`，边框 `1px solid #E0E0E0`，圆角 `4px`，内边距 `10px 12px`，字体 `14px`
- **聚焦状态**: 边框色 `#D4AF37`，边框宽度 `2px`，添加阴影 `0 0 0 3px rgba(212, 175, 55, 0.1)`
- **错误状态**: 边框色 `#F44336`，显示错误图标和错误提示文字
- **禁用状态**: 背景色 `#F8F5F0`，文字色 `#9E9E9E`，光标 `not-allowed`
- **只读状态**: 背景色 `#FAFAFA`，边框色 `#E0E0E0`

### 2.2 输入框变体

#### 带标签输入框
- **标签位置**: 输入框上方，间距 `8px`
- **标签样式**: 字体 `14px`，字重 `Medium`，颜色 `#2D2D2D`
- **必填标识**: 红色星号 `*`，位于标签后

#### 带图标输入框
- **前置图标**: 图标位于输入框左侧，尺寸 `20px`，颜色 `#9E9E9E`
- **后置图标**: 图标位于输入框右侧，用于清除、密码显示等操作
- **图标间距**: 与输入文字间距 `8px`

#### 搜索输入框
- **搜索图标**: 前置搜索图标，尺寸 `20px`
- **清除按钮**: 输入内容后显示清除按钮
- **快捷键提示**: 显示快捷键提示（如 `⌘K`）

### 2.3 文本域（Textarea）
- **最小高度**: `80px`
- **可调整大小**: 垂直方向可调整
- **字符计数**: 显示当前字符数/最大字符数
- **自动增长**: 根据内容自动调整高度（可选）

## 3. 选择器组件（Select）

### 3.1 下拉选择器

**视觉规范**:
- **默认状态**: 高度 `40px`，边框 `1px solid #E0E0E0`，圆角 `4px`，右侧下拉箭头图标
- **展开状态**: 边框色 `#D4AF37`，下拉面板显示，箭头图标旋转180度
- **选项悬停**: 背景色 `#F8F5F0`
- **选项选中**: 背景色 `rgba(212, 175, 55, 0.1)`，右侧显示勾选图标

### 3.2 多选选择器
- **已选标签**: 显示为标签形式，背景色 `#F8F5F0`，可点击删除
- **标签限制**: 超过3个标签显示"+N"
- **全选选项**: 下拉面板顶部显示"全选"选项

### 3.3 搜索选择器
- **搜索框**: 下拉面板顶部显示搜索框
- **实时过滤**: 输入时实时过滤选项
- **无结果提示**: 无匹配结果时显示"无匹配结果"

### 3.4 级联选择器
- **多级展示**: 支持多级选项展示
- **面包屑导航**: 显示当前选择路径
- **返回上级**: 支持返回上级选项

## 4. 复选框与单选按钮

### 4.1 复选框（Checkbox）

**视觉规范**:

- **悬停状态**: 边框色 `#D4AF37`
- **选中状态**: 背景色 `#D4AF37`，显示白色勾选图标
- **半选状态**: 背景色 `#D4AF37`，显示白色横线图标
- **禁用状态**: 背景色 `#F8F5F0`，边框色 `#E0E0E0`

### 4.2 单选按钮（Radio）

**视觉规范**:
- **默认状态**: 圆形 `16px × 16px`，边框 `1px solid #E0E0E0`
- **悬停状态**: 边框色 `#D4AF37`
- **选中状态**: 边框色 `#D4AF37`，内部显示金色圆点 `8px × 8px`
- **禁用状态**: 背景色 `#F8F5F0`，边框色 `#E0E0E0`

### 4.3 开关（Switch）

**视觉规范**:
- **默认状态**: 宽度 `44px`，高度 `24px`，背景色 `#E0E0E0`，圆形滑块 `20px × 20px`
- **开启状态**: 背景色 `#D4AF37`，滑块移至右侧
- **禁用状态**: 背景色 `#F8F5F0`，滑块颜色 `#9E9E9E`
