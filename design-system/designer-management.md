# 设计师管理模块界面设计

## 1. 模块概述

设计师管理模块提供团队成员管理、项目分配和能力标签功能，帮助企业高效管理设计团队。界面设计采用卡片和列表结合的布局，强调团队协作和人员信息的可视化展示。

## 2. 设计师列表页

### 2.1 页面布局

**整体结构**:
- **顶部操作栏**: 高度 `64px`，包含搜索、筛选、视图切换和邀请按钮
- **内容区域**: 卡片视图或列表视图，可切换
- **侧边筛选面板**: 宽度 `280px`，可折叠

### 2.2 顶部操作栏

**左侧区域**:
- **页面标题**: "设计师管理"，字体 `28px`，字重 `Bold`
- **统计信息**: "共 X 位成员"，字体 `14px`，颜色 `#9E9E9E`

**右侧区域**:
- **搜索框**: 宽度 `320px`，占位符"搜索姓名、职位、技能..."
- **筛选按钮**: 图标按钮，展开筛选面板
- **排序下拉**: 默认"姓名"，选项包括"加入时间"、"项目数量"
- **视图切换**: 卡片视图/列表视图切换按钮
- **邀请按钮**: 主要按钮，文字"邀请成员"，图标 `+`

### 2.3 卡片视图

**卡片尺寸**: 宽度自适应，高度 `320px`，3列（桌面端）、2列（平板端）、1列（移动端）

**卡片结构**:
1. **头像区域** (高度 `120px`):
   - 圆形头像，尺寸 `80px`，居中显示
   - 在线状态指示器（绿色/灰色圆点）
   - 背景渐变（深灰到金色）

2. **信息区域** (高度 `200px`，内边距 `16px`):
   - **姓名**: 字体 `18px`，字重 `Medium`，居中显示
   - **职位**: 字体 `14px`，颜色 `#9E9E9E`，居中显示
   - **部门**: 字体 `12px`，颜色 `#9E9E9E`
   - **技能标签**: 最多显示3个标签，超出显示"+N"
   - **统计数据**:
     - 负责项目：X个
     - 完成提案：X个
     - 团队评分：★★★★☆
   - **底部操作栏**:
     - 查看详情按钮
     - 发送消息按钮
     - 更多操作（三点图标）

**卡片状态**:
- **默认状态**: 背景 `#FFFFFF`，边框 `1px solid #E0E0E0`，阴影 `0 2px 8px rgba(0, 0, 0, 0.08)`
- **悬停状态**: 阴影加深 `0 4px 16px rgba(0, 0, 0, 0.12)`，头像轻微放大
- **离职状态**: 背景色变灰，显示"已离职"标签

### 2.4 列表视图

**表格列**:
- **成员**: 头像 + 姓名 + 在线状态
- **职位**: 职位名称
- **部门**: 部门名称
- **技能**: 技能标签（最多3个）
- **负责项目**: 项目数量
- **加入时间**: 日期
- **状态**: 在职/离职
- **操作**: 查看、编辑、禁用、删除

**表格样式**:
- **表头**: 背景色 `#F8F5F0`，字重 `Medium`，高度 `48px`
- **表格行**: 高度 `64px`，底部边框 `1px solid #E0E0E0`
- **悬停状态**: 背景色 `#FAFAFA`
- **选中状态**: 背景色 `rgba(212, 175, 55, 0.08)`

### 2.5 侧边筛选面板

**面板结构**:
- **部门筛选**: 多选复选框组
- **职位筛选**: 多选复选框组
- **技能筛选**: 标签云展示，支持多选
- **状态筛选**: 单选按钮组（全部/在职/离职）
- **项目负载**: 滑块（0-10个项目）
- **加入时间**: 日期范围选择器
- **底部操作**: "重置筛选"和"应用筛选"按钮

### 2.6 空状态

**无成员时**:
- **插图**: 空团队插图
- **提示文字**: "还没有团队成员，立即邀请第一位成员"
- **邀请按钮**: 主要按钮

**无搜索结果时**:
- **插图**: 搜索无结果插图
- **提示文字**: "未找到匹配的成员，请尝试其他关键词"
- **清除按钮**: 次要按钮

## 3. 设计师详情页

## 4. 邀请成员

### 4.1 邀请方式

**邀请模态框**:
- **标题**: "邀请团队成员"
- **邀请方式**: 标签页切换
  1. 邮件邀请
  2. 链接邀请
  3. 批量导入

### 4.2 邮件邀请

**表单字段**:
- **邮箱地址**: 文本输入框，支持输入多个邮箱（逗号分隔）
- **姓名**: 文本输入框（可选）
- **职位**: 下拉选择器
- **部门**: 下拉选择器
- **角色**: 下拉选择器（管理员/编辑者/查看者）
- **邀请消息**: 文本域，自定义邀请消息

**发送按钮**: 主要按钮，文字"发送邀请"

**邀请记录**:
- **表格展示**: 邮箱、姓名、发送时间、状态、操作
- **状态**: 待接受/已接受/已拒绝/已过期
- **操作**: 重新发送、撤销邀请

### 4.3 链接邀请

**邀请链接**:
- **生成链接**: 按钮，生成邀请链接
- **链接显示**: 文本框显示邀请链接
- **复制按钮**: 复制链接到剪贴板
- **二维码**: 显示邀请链接的二维码

**链接设置**:
- **有效期**: 下拉选择器（1天/7天/30天/永久）
- **使用次数**: 数字输入框（限制使用次数）
- **默认角色**: 下拉选择器

**链接管理**:
- **链接列表**: 显示所有生成的邀请链接
- **操作**: 复制、禁用、删除

### 4.4 批量导入

**导入方式**:
- **Excel导入**: 上传Excel文件
- **CSV导入**: 上传CSV文件
- **模板下载**: 下载导入模板

**导入步骤**:
1. **上传文件**: 拖拽上传或选择文件
2. **字段映射**: 映射Excel列与系统字段
3. **数据预览**: 预览导入数据
4. **确认导入**: 确认并导入数据

**导入结果**:
- **成功数量**: 显示成功导入的数量
- **失败数量**: 显示失败的数量
- **错误列表**: 显示失败的记录和原因
- **导出错误**: 按钮，导出错误记录

## 9. 响应式设计

### 9.1 桌面端 (>1024px)
- 3列卡片网格
- 完整功能展示
- 侧边筛选面板常驻

### 9.2 平板端 (768px-1024px)
- 2列卡片网格
- 侧边筛选面板可折叠
- 功能适当简化

### 9.3 移动端 (<768px)
- 1列卡片网格或列表视图
- 筛选面板改为底部抽屉
- 简化操作流程

## 10. 交互细节

### 10.1 微交互
- **卡片悬停**: 阴影加深，头像放大
- **在线状态**: 实时更新在线状态
- **操作反馈**: Toast提示操作结果

### 10.2 动画效果
- **页面切换**: 淡入淡出
- **模态框**: 从中心缩放出现
- **列表加载**: 从上到下依次淡入

### 10.3 快捷键
- `Ctrl/Cmd + N`: 邀请新成员
- `Ctrl/Cmd + F`: 搜索成员
- `Ctrl/Cmd + E`: 编辑选中成员
- `Esc`: 关闭模态框

## 11. 可访问性

### 11.1 键盘导航
- 所有交互元素支持Tab键导航
- 焦点指示器清晰可见
- Enter/Space键触发操作

### 11.2 屏幕阅读器
- 成员卡片提供完整描述
- 按钮提供aria-label
- 表单元素关联label

### 11.3 色彩对比
- 文本与背景对比度≥4.5:1
- 状态指示使用图标+文字双重标识
- 支持高对比度模式
