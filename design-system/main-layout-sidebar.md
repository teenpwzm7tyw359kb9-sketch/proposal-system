## 1. 主应用布局

### 1.1 整体布局结构

```
┌─────────────────────────────────────────────────────────────────┐
│                        顶部导航栏 (64px)                         │
│  [Logo] [搜索框]                    [通知] [用户头像] [设置]    │
└─────────────────────────────────────────────────────────────────┘
┌──────────┬──────────────────────────────────────────────────────┐
│          │                                                      │
│          │                                                      │
│  侧边栏   │                  主内容区域                          │
│ (240px)  │              (动态内容展示区)                        │
│          │                                                      │
│          │                                                      │
│          │                                                      │
└──────────┴──────────────────────────────────────────────────────┘
```

### 1.2 布局尺寸规范

**顶部导航栏**：
- 高度：`64px`
- 背景色：`#FFFFFF`
- 边框：底部 `1px solid #F1F1F1`
- 阴影：`0 1px 3px rgba(0, 0, 0, 0.05)`
- 固定定位，始终可见

**侧边栏**：
- 宽度：`240px`（展开）/ `64px`（收起）
- 背景色：`#FAFAFA`
- 边框：右侧 `1px solid #F1F1F1`
- 固定定位，全高度

**主内容区域**：
- 宽度：`calc(100vw - 240px)`（侧边栏展开）
- 背景色：`#FAF9F6`
- 内边距：`24px`
- 最大宽度：`1440px`（居中显示）

### 2.2 侧边栏导航项设计

```

- 内边距：`12px 16px`

- 文字颜色：`#2D2D2D`
- 文字大小：`14px`
- 圆角：`6px`

**悬停状态**：
```

│  📊  仪表板           │  ← 背景色变化

```
- 背景色：`#F8F5F0`

- 文字颜色：`#D4AF37`
- 过渡动画：`0.2s ease`

**激活状态**（当前页面）：
```

```

- 左侧边框：`3px solid #D4AF37`
- 图标颜色：`#D4AF37`
- 文字颜色：`#D4AF37`
- 文字字重：`600`

### 2.3 侧边栏收起状态

**收起后（宽度 64px）**：
```
┌────┐
│ 📊 │  ← 仅显示图标

**悬停提示**：

- 提示位置：右侧
- 提示背景：`#2D2D2D`
- 提示文字：`#FFFFFF`
- 提示延迟：`300ms`

### 2.4 侧边栏展开/收起按钮

**位置**：侧边栏底部
**图标**：`«` (收起) / `»` (展开)
**样式**：
