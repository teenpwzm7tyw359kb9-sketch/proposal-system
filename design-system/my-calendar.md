# 我的日历模块界面设计

## 1. 模块概述

我的日历模块提供个人和团队日程管理功能，支持多种视图、事件管理和日历同步。界面设计强调时间可视化、团队协作和高效的日程安排。

## 2. 日历主页

### 2.1 页面布局

**整体结构**:
- **顶部操作栏**: 高度 `64px`，包含日期导航、视图切换、搜索和创建按钮
- **左侧日历列表**: 宽度 `240px`，显示个人日历、团队日历、订阅日历
- **主要内容区域**: 根据视图类型动态调整
- **右侧事件详情**: 宽度 `320px`，显示选中事件的详细信息

### 2.2 顶部操作栏

**左侧区域**:
- **页面标题**: "我的日历"，字体 `28px`，字重 `Bold`
- **日期导航**:
  - 左右箭头按钮（切换日期）
  - 今天按钮（快速回到今天）
  - 当前日期显示（可点击打开日期选择器）

**右侧区域**:
- **视图切换**: 按钮组（日/周/月/年视图）
- **搜索框**: 宽度 `240px`，占位符"搜索事件..."
- **设置按钮**: 图标按钮，打开日历设置
- **创建按钮**: 主要按钮，文字"创建事件"，图标 `+`

### 2.3 左侧日历列表

**我的日历**:
- **个人日历**: 默认日历，颜色 `#D4AF37`（金色）
- **工作日历**: 工作相关事件，颜色 `#2196F3`（蓝色）
- **个人事务**: 私人事件，颜色 `#4CAF50`（绿色）
- **显示/隐藏**: 每个日历前有复选框，控制显示/隐藏

**团队日历**:
- **设计部**: 团队共享日历
- **销售部**: 团队共享日历
- **管理层**: 管理层日历
- **权限标识**: 显示权限图标（只读/可编辑）

**订阅日历**:
- **中国节假日**: 系统预设日历
- **行业活动**: 订阅的行业活动日历
- **添加订阅**: 按钮，添加新的订阅日历

**日历管理**:
- **创建日历**: 文本按钮，创建新日历
- **导入日历**: 文本按钮，导入外部日历
- **日历设置**: 文本按钮，打开设置页面

### 2.4 日历颜色系统

**颜色分配**:
- **个人日历**: 金色 `#D4AF37`
- **工作日历**: 蓝色 `#2196F3`
- **团队日历**: 紫色 `#9C27B0`
- **节假日**: 红色 `#F44336`
- **生日**: 粉色 `#E91E63`
- **自定义**: 用户可选择12种预设颜色

## 3. 日视图

### 3.1 时间轴布局

**全天事件区域** (顶部):
- **高度**: 根据全天事件数量动态调整
- **事件卡片**: 横向排列，显示事件标题和日历颜色
- **折叠功能**: 超过3个全天事件时显示"展开更多"

**时间轴区域**:
- **时间刻度**: 00:00-23:59，每小时一个主刻度，每30分钟一个次刻度
- **当前时间线**: 红色水平线 + 圆点，实时移动
- **工作时间高亮**: 09:00-18:00背景色略深，突出工作时间

### 3.2 事件显示

**事件卡片样式**:
- **背景色**: 使用日历颜色，透明度 `0.2`
- **左侧边框**: 宽度 `4px`，使用日历颜色
- **文字颜色**: 深灰色 `#2D2D2D`
- **圆角**: `4px`
- **内边距**: `4px 8px`

**事件信息**:
- **时间**: 10:00-11:00（小字，灰色）
- **标题**: 客户会议（中字，黑色）
- **地点**: 会议室A（小字，灰色，前置地点图标）
- **参与者**: 显示参与者头像（最多3个，超出显示"+N"）

**事件状态指示**:
- **待确认**: 虚线边框
- **已确认**: 实线边框
- **已完成**: 背景色变浅，文字添加删除线
- **已取消**: 背景色变灰，文字添加删除线

### 3.3 交互功能

**创建事件**:
- **点击空白时间**: 弹出快速创建表单
- **拖拽创建**: 在时间轴上拖拽选择时间段
- **双击创建**: 双击空白时间打开完整创建表单

**编辑事件**:
- **点击事件**: 右侧显示事件详情
- **拖拽移动**: 拖拽事件卡片调整时间
- **拖拽调整**: 拖拽事件卡片边缘调整时长
- **双击编辑**: 双击事件打开编辑表单

**右键菜单**:
- 查看详情
- 编辑事件
- 复制事件
- 删除事件
- 更改日历
- 发送邀请

## 4. 周视图

### 4.1 网格布局

**表头设计**:
- **星期**: 周一、周二、周三、周四、周五、周六、周日
- **日期**: 对应的具体日期（1月1日、1月2日...）
- **今天标识**: 金色圆形背景 + 白色文字
- **周末标识**: 浅红色背景

**时间网格**:
- **左侧时间列**: 宽度 `60px`，显示时间刻度
- **七列内容区**: 每列代表一天，宽度自适应
- **网格线**: 浅灰色虚线，每小时一条

### 4.2 事件展示

**单日事件**: 在对应日期列中事件**: 横跨多个日期列，显示在全天事件区域
**重叠事件**: 垂直堆叠，宽度按比例缩小

**事件密度处理**:
- **正常密度**: 完整显示事件信息
- **高密度**: 仅显示时间和标题
- **超高密度**: 显示为彩色条形 + 数字标识

### 4.3 团队视图

**团队成员列表** (左侧):
- **成员头像**: 圆形头像，尺寸 `32px`
- **成员姓名**: 字体 `14px`
- **在线状态**: 绿色/灰色圆点
- **忙闲状态**: 忙碌/空闲/离开

**成员日程展示**:
- **横向时间轴**: 显示每个成员的日程安排
- **忙碌时段**: 彩色条形显示
- **空闲时段**: 空白或浅灰色
- **冲突检测**: 自动标识时间冲突

**会议安排助手**:
- **查找空闲时间**: 自动查找所有成员都空闲的时间段
- **建议时间**: 高亮显示建议的会议时间
- **一键创建**: 点击建议时间快速创建团队会议

**月份导航**:

- **左右箭头**: 切换月份

**日期单元格**:

## 7. 事件创建/编辑

### 7.1 快速创建表单

**表单字段** (模态框，宽度 `400px`):
- **事件标题**: 必填，文本输入框
- **日期时间**: 必填，日期时间选择器
- **日历**: 下拉选择器，选择归属日历
- **创建按钮**: 主要按钮
- **更多选项**: 文本按钮，打开完整表单

### 7.2 完整创建表单

**基本信息**:
- **事件标题**: 必填，文本输入框
- **日期时间**: 必填，开始时间 + 结束时间
- **全天事件**: 开关
- **重复**: 下拉选择器（不重复/每天/每周/每月/每年/自定义）
- **日历**: 下拉选择器
- **颜色**: 颜色选择器（可覆盖日历默认颜色）

**地点信息**:
- **地点**: 文本输入框，支持地图搜索
- **会议室**: 下拉选择器，显示可用会议室
- **线上会议**: 开关 + 会议链接输入

**参与者**:
- **添加参与者**: 搜索选择器，支持多选
- **参与者列表**: 显示已添加的参与者
- **参与者状态**: 待确认/已接受/已拒绝/待定
- **发送邀请**: 开关，自动发送邮件邀请

**提醒设置**:
- **提醒时间**: 多选（不提醒/事件开始时/5分钟前/15分钟前/30分钟前/1小时前/1天前/自定义）
- **提醒方式**: 多选（系统通知/邮件/短信/微信）

**描述和附件**:
- **描述**: 富文本编辑器
- **附件**: 文件上传器，支持拖拽上传
- **链接**: 添加相关链接

**高级选项**:
- **忙闲状态**: 单选（忙碌/空闲/暂定/外出）
- **可见性**: 单选（公开/私密）
- **访客权限**: 单选（可修改事件/仅查看）

### 7.3 重复事件设置

**重复模式**:
- **每天**: 每N天重复
- **每周**: 选择星期几（多选）
- **每月**: 选择每月第几天或第几个星期几
- **每年**: 选择月份和日期

**结束条件**:
- **永不结束**: 单选
- **结束日期**: 单选 + 日期选择器
- **重复次数**: 单选 + 数字输入框

**例外日期**:
- **添加例外**: 选择不重复的日期
- **例外列表**: 显示所有例外日期

## 10. 团队协作功能

### 10.1 共享日历

**创建共享日历**:
- **日历名称**: 文本输入框
- **日历描述**: 文本域
- **日历颜色**: 颜色选择器
- **成员**: 添加团队成员
- **权限**: 为每个成员设置权限

**权限级别**:
- **所有者**: 完全控制权限
- **编辑者**: 可创建、编辑、删除事件
- **查看者**: 仅可查看事件
- **忙闲查看**: 仅可查看忙闲状态

### 10.2 会议邀请

**发送邀请**:
- **邀请方式**: 邮件/系统通知/企业微信/钉钉
- **邀请内容**: 自动生成邀请内容
- **附加信息**: 会议议程、准备材料

**接收邀请**:
- **邀请通知**: 系统通知 + 邮件通知
- **快速响应**: 接受/拒绝/待定
- **添加到日历**: 自动添加到个人日历

**邀请状态追踪**:

- **已拒绝**: 红色标识

### 10.3 冲突检测

**自动检测**: 创建事件时自动检测时间冲突

**建议时间**: 提供无冲突的替代时间
**强制创建**: 允许用户忽略冲突强制创建

### 12.1 微交互
- **事件拖拽**: 平滑的拖拽动画
- **时间选择**: 时间轴滚动动画
- **状态变更**: 颜色渐变动画

### 12.2 动画效果

- **事件创建**: 从点击位置放大出现

## 13. 可访问性

### 13.1 键盘导航
- 所有交互元素支持Tab键导航
- 方向键支持日历导航
- Enter/Space键触发操作

### 13.2 屏幕阅读器
- 日历单元格提供完整描述
- 事件提供详细信息
- 按钮提供aria-label

### 13.3 色彩对比
- 文本与背景对比度≥4.5:1
- 事件使用多重标识（颜色+图标+文字）
- 支持高对比度模式
