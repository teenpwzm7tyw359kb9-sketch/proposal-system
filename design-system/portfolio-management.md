# 作品集管理模块界面设计

## 1. 模块概述

作品集管理模块用于展示和管理设计案例作品，支持公共展示和SEO优化。界面设计采用杂志风格的瀑布流布局，强调视觉冲击力和专业性，为设计公司提供高品质的作品展示平台。

## 2. 作品集列表页

### 2.1 页面布局

**整体结构**:
- **顶部操作栏**: 高度 `64px`，包含搜索、筛选、排序和创建按钮
- **内容区域**: 瀑布流布局，3列（桌面端）、2列（平板端）、1列（移动端）
- **侧边筛选面板**: 宽度 `280px`，可折叠，包含分类、标签、风格等筛选条件

### 2.2 顶部操作栏

**左侧区域**:
- **页面标题**: "作品集管理"，字体 `28px`，字重 `Bold`，颜色 `#2D2D2D`
- **统计信息**: 显示"共 X 个作品"，字体 `14px`，颜色 `#9E9E9E`

**右侧区域**:
- **搜索框**: 宽度 `320px`，前置搜索图标，占位符"搜索项目名称、客户名称..."
- **筛选按钮**: 图标按钮，点击展开筛选面板
- **排序下拉**: 默认"最近更新"，选项包括"创建时间"、"项目名称"、"浏览量"
- **视图切换**: 瀑布流/网格视图切换按钮
- **创建按钮**: 主要按钮，文字"创建作品"，图标 `+`

### 2.3 作品卡片设计

**卡片尺寸**: 宽度自适应，高度根据内容动态调整

**卡片结构**:
1. **图片区域** (高度 `240px-400px`):
   - 显示作品主图，宽高比根据图片自适应
   - 支持多图轮播，显示小圆点指示器
   - 悬停时显示快速预览按钮

2. **信息区域** (内边距 `16px`):
   - **项目标题**: 字体 `16px`，字重 `Medium`，最多显示2行
   - **项目描述**: 字体 `14px`，颜色 `#9E9E9E`，最多显示3行
   - **项目标签**: 标签云展示，最多显示3个，超出显示"+N"
   - **统计数据**: 浏览量、点赞数、评论数
   - **底部操作栏**:
     - 左侧：访问状态（公开/私有/指定客户）
     - 右侧：编辑按钮、分享按钮、删除按钮

**卡片状态**:
- **默认状态**: 背景 `#FFFFFF`，边框 `1px solid #E0E0E0`，阴影 `0 2px 8px rgba(0, 0, 0, 0.08)`
- **悬停状态**: 阴影加深 `0 4px 16px rgba(0, 0, 0, 0.12)`，图片轻微放大
- **选中状态**: 边框色 `#D4AF37`，边框宽度 `2px`

### 2.4 侧边筛选面板

**面板结构**:
- **项目类型**: 全部/住宅/商业/办公/酒店/其他
- **设计风格**: 现代简约/新中式/欧式古典/北欧/工业风/日式/其他
- **空间类型**: 客厅/卧室/厨房/卫生间/办公室/餐厅/其他
- **预算范围**: 价格区间选择器
- **标签筛选**: 标签云展示，支持多选
- **创建人筛选**: 下拉选择器，显示团队成员
- **底部操作**: "重置筛选"和"应用筛选"按钮

### 2.5 空状态

**无作品时**:
- **插图**: 中央显示空画廊插图
- **提示文字**: "还没有作品，立即创建第一个作品集"
- **创建按钮**: 主要按钮，文字"创建作品"

**无搜索结果时**:
- **插图**: 搜索无结果插图
- **提示文字**: "未找到匹配的作品，请尝试其他关键词"
- **清除按钮**: 次要按钮，文字"清除搜索条件"

## 3. 作品创建/编辑页

### 3.1 基本信息设置

**表单布局** (居中，最大宽度 `600px`):
- **项目名称**: 必填，文本输入框，占位符"请输入项目名称"
- **项目副标题**: 可选，文本输入框，占位符"项目副标题或标语"
- **项目类型**: 必填，下拉选择器
- **设计风格**: 必填，多选下拉
- **项目地址**: 可选，文本输入框
- **项目面积**: 可选，数字输入框 + 单位选择
- **项目预算**: 可选，数字输入框
- **项目周期**: 可选，日期范围选择器
- **项目描述**: 必填，富文本编辑器，支持图文混排
- **项目标签**: 可选，标签选择器，支持自定义
- **访问权限**: 单选按钮组（公开/私有/指定客户可见）

### 3.2 图片上传与管理

**图片上传区**:
- **拖拽上传**: 虚线边框区域，支持拖拽上传
- **选择文件**: 按钮，支持多选
- **批量上传**: 支持同时上传多张图片

**图片管理区**:
- **缩略图网格**: 显示所有已上传图片
- **图片操作**:
  - 点击设为主图
  - 拖拽调整顺序
  - 编辑图片信息（标题、描述、版权信息）
  - 删除图片
- **图片信息编辑**:
  - 图片标题
  - 图片描述
  - 版权信息
  - 拍摄时间
  - 拍摄设备

### 3.3 多媒体支持

**视频上传**:
- 支持MP4、MOV格式
- 显示视频封面 + 播放按钮
- 视频信息编辑（标题、描述、时长）

**360°全景**:
- 支持equirectangular格式
- 预览模式，支持鼠标拖拽查看
- 移动端支持陀螺仪控制

**VR展示**:
- 支持WebVR格式
- 提示用户使用VR设备查看
- 提供传统2D替代方案

### 3.4 项目详情页编辑

**页面结构**:
- **项目背景**: 富文本编辑器
- **设计理念**: 富文本编辑器
- **实施过程**: 时间轴编辑器
- **最终效果**: 图片/视频展示
- **客户评价**: 引用编辑器
- **技术参数**: 表格编辑器

**编辑器功能**:
- **富文本**: 支持加粗、斜体、链接、列表、引用等
- **图片插入**: 支持从已上传图片中选择
- **视频插入**: 支持嵌入YouTube、Bilibili等
- **代码块**: 支持技术参数展示
- **分隔线**: 支持多种分隔线样式

## 4. 作品详情页（内部管理）

### 4.1 页面布局

**顶部信息卡**:
- **项目主图**: 全宽展示，支持轮播
- **项目标题**: 字体 `32px`，字重 `Bold`
- **项目信息**: 项目类型、风格、面积、预算等
- **操作按钮**: 编辑、分享、删除、预览

**标签页导航**:
1. 项目详情
2. 图片管理
3. 访问统计
4. 分享记录

### 4.2 项目详情标签页

**内容展示**:
- **富文本内容**: 按照编辑时的格式展示
- **多媒体内容**: 图片、视频、360°全景正常展示
- **交互元素**: 支持点赞、评论、分享

### 4.3 图片管理标签页

**图片网格**: 所有图片缩略图展示
**批量操作**: 选择多张图片进行批量操作
**操作选项**: 设为主图、编辑信息、重新排序、删除

### 4.4 访问统计标签页

**统计概览**:
- 总访问次数
- 独立访客数
- 平均停留时间
- 社交分享次数

**访问趋势**: 折线图展示最近30天访问趋势
**来源分析**: 饼图展示访问来源（直接访问、搜索引擎、社交媒体等）
**热门页面**: 显示最受欢迎的图片/视频

### 4.5 分享记录标签页

**分享渠道统计**: 各渠道分享次数
**分享记录列表**: 时间、渠道、链接、访问次数
**操作**: 复制链接、重新分享、撤销分享

## 5. 公共展示页（外部访问）

### 5.1 首页设计

**头部区域**:
- **Logo**: 企业Logo + 自定义域名
- **导航菜单**: 关于我们、作品集、服务、联系
- **搜索功能**: 作品搜索
- **语言切换**: 中文/英文切换

**作品展示区**:
- **瀑布流布局**: 动态加载更多作品
- **分类导航**: 顶部固定分类标签
- **筛选功能**: 按类型、风格、空间筛选
- **排序选项**: 按时间、热度、推荐排序

**底部区域**:
- **企业信息**: 公司介绍、联系方式
- **社交媒体**: 微信、微博、小红书等链接
- **版权信息**: 版权声明、备案信息

### 5.2 作品详情页（公共）

**页面布局**:
- **全屏图片展示**: 支持左右切换、缩放
- **项目信息侧边栏**: 固定右侧，显示项目基本信息
- **项目详情内容**: 滚动展示详细内容
- **相关作品推荐**: 底部显示相似作品

**交互功能**:
- **图片收藏**: 点赞功能
- **社交分享**: 一键分享到微信、微博、小红书
- **联系咨询**: 底部固定咨询按钮
- **返回作品集**: 顶部导航

### 5.3 SEO优化功能

**URL结构**:
- 作品集首页: `portfolio.company.com`
- 作品详情页: `portfolio.company.com/works/project-name`
- 分类页面: `portfolio.company.com/category/residential`

**Meta标签**:
- 自动生成标题、描述、关键词
- 支持自定义覆盖
- Open Graph标签支持

**结构化数据**:
- 自动生成JSON-LD结构化数据
- 支持Google Rich Results
- 符合Schema.org标准

**性能优化**:
- 图片懒加载
- 代码分割
- CDN加速
- 移动端优化

## 6. 自定义域名设置

### 6.1 域名绑定

**设置页面**:
- **当前域名**: 显示当前绑定的域名
- **域名输入**: 文本输入框，支持输入自定义域名
- **验证步骤**:
  1. 在DNS提供商添加CNAME记录
  2. 系统自动验证
  3. 绑定成功

**SSL证书**:
- 自动申请Let's Encrypt证书
- HTTPS强制启用
- 证书到期自动续期

### 6.2 品牌定制

**Logo设置**:
- 上传企业Logo
- 设置Logo尺寸和位置
- 预览效果

**配色方案**:
- 主色调设置（建议使用企业VI色）
- 辅助色设置
- 实时预览

**字体选择**:
- 中文字体选择
- 英文字体选择
- 字号设置

**页脚定制**:
- 企业信息编辑
- 联系方式设置
- 社交媒体链接
- 版权信息

## 7. 响应式设计

### 7.1 桌面端 (>1024px)
- 3列瀑布流
- 完整功能展示
- 侧边信息栏常驻

### 7.2 平板端 (768px-1024px)
- 2列瀑布流
- 信息栏折叠为抽屉
- 功能适当简化

### 7.3 移动端 (<768px)
- 1列瀑布流
- 导航改为底部tab
- 图片优先加载
- 触摸优化

## 8. 交互细节

### 8.1 图片交互
- **悬停效果**: 图片轻微放大，显示快速操作按钮
- **点击效果**: 全屏查看，支持手势缩放
- **加载效果**: 渐进式加载，模糊到清晰

### 8.2 动画效果
- **页面切换**: 淡入淡出
- **图片加载**: 从中心放大
- **滚动效果**: 视差滚动，增强层次感

### 8.3 社交分享
- **微信分享**: 生成微信小程序码
- **微博分享**: 自动带话题标签
- **小红书分享**: 适配小红书图片比例

## 9. 可访问性

### 9.1 图片可访问性
- 所有图片提供alt文本
- 支持键盘导航图片
- 屏幕阅读器友好

### 9.2 色彩对比
- 文字与背景对比度≥4.5:1
- 状态指示使用多重标识

### 9.3 键盘导航
- 支持Tab键导航所有交互元素
- 方向键支持图片切换
- Enter键触发主要操作
