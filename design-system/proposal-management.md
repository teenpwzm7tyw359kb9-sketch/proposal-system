# 提案管理模块界面设计

## 1. 模块概述

提案管理是系统的核心模块，用于创建、编辑、分享和分析提案。界面设计强调专业性、效率和视觉美感，采用杂志风格布局，突出内容展示。

## 2. 提案列表页

### 2.1 页面布局

**整体结构**:
- **顶部操作栏**: 高度 `64px`，包含搜索、筛选、排序和创建按钮
- **内容区域**: 采用卡片网格布局，3列（桌面端）、2列（平板端）、1列（移动端）
- **侧边筛选面板**: 宽度 `280px`，可折叠，包含状态、日期、标签等筛选条件

### 2.2 顶部操作栏

**左侧区域**:
- **页面标题**: "提案管理"，字体 `28px`，字重 `Bold`，颜色 `#2D2D2D`
- **统计信息**: 显示"共 X 个提案"，字体 `14px`，颜色 `#9E9E9E`

**右侧区域**:
- **搜索框**: 宽度 `320px`，前置搜索图标，占位符"搜索提案名称、客户名称..."
- **筛选按钮**: 图标按钮，点击展开筛选面板
- **排序下拉**: 默认"最近更新"，选项包括"创建时间"、"名称"、"客户名称"
- **视图切换**: 网格视图/列表视图切换按钮
- **创建按钮**: 主要按钮，文字"创建提案"，图标 `+`

### 2.3 提案卡片设计

**卡片尺寸**: 宽度自适应，高度 `360px`

**卡片结构**:
1. **封面图区域** (高度 `200px`):
   - 显示提案封面图，宽高比 `16:9`
   - 无封面时显示默认渐变背景（深灰到金色渐变）
   - 左上角显示状态标签（草稿/已发布/已归档）
   - 右上角显示操作菜单按钮（三点图标）

2. **信息区域** (高度 `160px`，内边距 `16px`):
   - **提案标题**: 字体 `16px`，字重 `Medium`，最多显示2行，超出省略
   - **客户名称**: 字体 `14px`，颜色 `#9E9E9E`，前置客户图标
   - **创建时间**: 字体 `12px`，颜色 `#9E9E9E`，前置时钟图标
   - **标签**: 最多显示3个标签，超出显示"+N"
   - **底部操作栏**:
     - 左侧：查看次数图标 + 数字
     - 右侧：编辑按钮、分享按钮、删除按钮（图标按钮）

**卡片状态**:
- **默认状态**: 背景 `#FFFFFF`，边框 `1px solid #E0E0E0`，阴影 `0 2px 8px rgba(0, 0, 0, 0.08)`
- **悬停状态**: 阴影加深 `0 4px 16px rgba(0, 0, 0, 0.12)`，封面图轻微放大（scale 1.05）
- **选中状态**: 边框色 `#D4AF37`，边框宽度 `2px`

### 2.4 侧边筛选面板

**面板结构**:
- **状态筛选**: 全部/草稿/已发布/已归档，使用单选按钮组
- **日期范围**: 日期选择器，支持快捷选项（今天、本周、本月、自定义）
- **客户筛选**: 搜索选择器，支持多选
- **标签筛选**: 标签云展示，点击选中/取消
- **创建人筛选**: 下拉选择器，显示团队成员列表
- **底部操作**: "重置筛选"和"应用筛选"按钮

### 2.5 空状态

**无提案时**:
- **插图**: 中央显示空文件夹插图，尺寸 `200px × 200px`
- **提示文字**: "还没有提案，立即创建第一个提案"
- **创建按钮**: 主要按钮，文字"创建提案"

**无搜索结果时**:
- **插图**: 搜索无结果插图
- **提示文字**: "未找到匹配的提案，请尝试其他关键词"
- **清除按钮**: 次要按钮，文字"清除搜索条件"

## 3. 提案创建页

### 3.1 创建流程

**步骤条** (顶部固定):
1. 选择模板
2. 基本信息
3. 内容编辑
4. 预览发布

### 3.2 步骤1：选择模板

**页面布局**:
- **标题**: "选择提案模板"，居中显示
- **模板分类**: 顶部标签页，分类包括"全部"、"住宅设计"、"商业空间"、"办公空间"、"软装方案"
- **模板网格**: 3列布局，每个模板卡片展示缩略图、名称、描述

**模板卡片**:
- **尺寸**: 宽度自适应，高度 `400px`
- **缩略图**: 高度 `280px`，显示模板预览图
- **信息区**:
  - 模板名称，字体 `16px`，字重 `Medium`
  - 模板描述，字体 `14px`，颜色 `#9E9E9E`，最多2行
  - 使用次数，字体 `12px`
- **操作**: "使用此模板"按钮，悬停时显示"预览模板"按钮

**空白模板选项**:
- 第一个卡片为"从空白开始"，显示加号图标
- 点击直接进入内容编辑

### 3.3 步骤2：基本信息

**表单布局** (居中，最大宽度 `600px`):
- **提案名称**: 必填，文本输入框，占位符"请输入提案名称"
- **客户名称**: 必填，搜索选择器，支持新建客户
- **项目地址**: 可选，文本输入框
- **项目类型**: 必填，下拉选择器（住宅/商业/办公/其他）
- **项目面积**: 可选，数字输入框，单位"㎡"
- **预算范围**: 可选，区间选择器
- **项目描述**: 可选，文本域，最多500字
- **标签**: 可选，标签选择器，支持自定义标签

**底部操作**:
- **上一步**: 次要按钮
- **下一步**: 主要按钮

### 3.4 步骤3：内容编辑

详见"画布编辑器设计"文档

### 3.5 步骤4：预览发布

**预览区域**:
- **全屏预览**: 模拟客户查看效果
- **设备切换**: 桌面/平板/移动端预览切换
- **页面导航**: 左侧显示页面缩略图导航

**发布设置** (右侧面板，宽度 `360px`):
- **发布状态**: 单选按钮组（草稿/立即发布）
- **访问控制**:
  - 密码保护：开关 + 密码输入框
  - 有效期：日期选择器（1天/7天/30天/永久）
  - 水印设置：开关 + 水印文字输入
- **分享设置**:
  - 允许下载：开关
  - 允许评论：开关
  - 查看统计：开关（默认开启）
- **通知设置**:
  - 通知客户：开关 + 邮件/短信/微信选择
  - 通知团队：开关 + 成员多选

**底部操作**:
- **保存草稿**: 次要按钮
- **发布提案**: 主要按钮

## 4. 提案编辑页

### 4.1 页面布局

**顶部工具栏** (高度 `56px`，背景 `#FFFFFF`，固定定位):
- **左侧**: 返回按钮、提案名称（可点击编辑）、自动保存状态
- **中间**: 撤销/重做按钮、页面导航下拉
- **右侧**: 预览按钮、分享按钮、发布按钮

**主编辑区域**:
- **左侧工具面板**: 宽度 `280px`，包含页面管理、模块库、素材库
- **中央画布区**: 宽度自适应，显示当前编辑页面
- **右侧属性面板**: 宽度 `320px`，显示选中元素的属性设置

详细设计见"画布编辑器设计"文档

## 5. 提案详情页

### 5.1 页面布局

**顶部信息卡** (背景渐变，深灰到金色):
- **提案标题**: 字体 `32px`，字重 `Bold`，颜色 `#FFFFFF`
- **客户信息**: 客户名称、项目地址、项目类型
- **状态标签**: 右上角显示当前状态
- **操作按钮**: 编辑、分享、下载、删除

**标签页导航**:
1. 提案内容
2. 分享记录
3. 访问统计
4. 版本历史

### 5.2 提案内容标签页

**页面预览**:
- **缩略图网格**: 显示所有页面的缩略图
- **点击查看**: 点击缩略图全屏查看该页面
- **页面信息**: 显示页面标题、页面类型

### 5.3 分享记录标签页

**分享列表**:
- **表格展示**: 分享时间、分享方式、访问链接、访问次数、有效期、操作
- **操作**: 复制链接、重新分享、撤销分享
- **筛选**: 按分享方式、时间范围筛选

### 5.4 访问统计标签页

**统计概览** (4个统计卡片):
- 总访问次数
- 独立访客数
- 平均停留时间
- 完成查看率

**访问趋势图**:
- 折线图，显示最近30天的访问趋势
- 可切换查看访问次数/独立访客数

**页面热力图**:
- 显示每个页面的访问次数和停留时间
- 使用颜色深浅表示热度

**访客列表**:
- 表格展示：访客IP、访问时间、停留时间、查看页数、设备类型
- 支持导出Excel

### 5.5 版本历史标签页

**版本列表**:
- **时间轴展示**: 左侧时间轴，右侧版本信息
- **版本信息**: 版本号、修改时间、修改人、修改说明
- **操作**: 查看、恢复、对比

**版本对比**:
- 左右分屏显示两个版本
- 高亮显示差异部分

## 6. 提案分享页（客户视角）

### 6.1 访问验证

**密码保护页**:
- **居中卡片**: 最大宽度 `400px`
- **Logo**: 顶部显示企业Logo
- **提示文字**: "此提案受密码保护，请输入访问密码"
- **密码输入框**: 密码输入框 + 查看按钮
- **提交按钮**: 主要按钮，文字"访问提案"

**过期提示页**:
- **提示图标**: 时钟图标
- **提示文字**: "此提案链接已过期，请联系设计师获取新链接"
- **联系方式**: 显示设计师联系方式

### 6.2 提案查看页

**顶部导航栏** (高度 `64px`，半透明背景):
- **左侧**: 企业Logo、提案标题
- **右侧**: 页面导航、下载按钮（如允许）、评论按钮

**内容展示区**:
- **全屏展示**: 每页全屏显示，滚动切换
- **页面指示器**: 右侧显示当前页码/总页数
- **导航箭头**: 左右箭头切换页面
- **水印**: 如设置水印，在页面右下角显示

**底部操作栏** (固定底部):
- **页面缩略图**: 横向滚动显示所有页面缩略图
- **快速跳转**: 点击缩略图快速跳转

### 6.3 评论功能

**评论面板** (右侧抽屉，宽度 `400px`):
- **评论列表**: 显示所有评论，按时间倒序
- **评论卡片**:
  - 评论人头像、姓名
  - 评论内容
  - 评论时间
  - 回复按钮
- **发表评论**: 底部固定输入框 + 发送按钮

**页面标注**:
- 点击页面任意位置添加标注
- 标注显示为金色圆点 + 序号
- 点击标注显示评论内容

## 7. 响应式设计

### 7.1 桌面端 (>1024px)
- 3列卡片网格
- 侧边筛选面板常驻
- 完整功能展示

### 7.2 平板端 (768px-1024px)
- 2列卡片网格
- 侧边筛选面板可折叠
- 操作按钮适当简化

### 7.3 移动端 (<768px)
- 1列卡片网格
- 筛选面板改为底部抽屉
- 操作按钮改为底部固定操作栏
- 提案编辑功能简化，建议使用桌面端

## 8. 交互细节

### 8.1 微交互
- **卡片悬停**: 阴影加深，封面图轻微放大
- **按钮点击**: 涟漪效果，颜色加深
- **加载状态**: 骨架屏 + 旋转加载器
- **操作反馈**: Toast提示成功/失败

### 8.2 动画效果
- **页面切换**: 淡入淡出，持续时间 `300ms`
- **模态框**: 从中心缩放出现，持续时间 `250ms`
- **抽屉**: 从侧边滑入，持续时间 `300ms`
- **列表加载**: 从上到下依次淡入，间隔 `50ms`

### 8.3 快捷键
- `Ctrl/Cmd + N`: 创建新提案
- `Ctrl/Cmd + S`: 保存提案
- `Ctrl/Cmd + P`: 预览提案
- `Ctrl/Cmd + K`: 打开搜索
- `Esc`: 关闭模态框/抽屉

## 9. 可访问性

### 9.1 键盘导航
- 所有交互元素支持Tab键导航
- 焦点指示器清晰可见
- 支持Enter/Space键触发操作

### 9.2 屏幕阅读器
- 所有图片提供alt文本
- 按钮提供aria-label
- 表单元素关联label

### 9.3 色彩对比
- 文本与背景对比度≥4.5:1
- 状态标签使用图标+文字双重标识
