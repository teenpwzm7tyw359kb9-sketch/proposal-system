# 提案展示系统产品需求文档（PRD）

## 1. 执行摘要

### 1.1 项目背景
随着中国设计行业、全屋定制行业和软装行业的快速发展，企业对专业提案展示工具的需求日益增长。传统PPT和PDF格式已无法满足现代客户对交互式、个性化提案体验的期望。本提案展示系统旨在为设计公司、全屋定制公司和软装公司提供一体化的数字化提案解决方案。

### 1.2 产品愿景
打造中国领先的SaaS提案展示平台，帮助设计类企业提升提案效率、增强客户体验、优化业务流程，实现从提案到成交的全流程数字化管理。

### 1.3 核心价值
- **提升提案效率**：通过模板化、模块化设计，减少重复工作
- **增强客户体验**：提供交互式、多媒体提案展示，提升专业形象
- **数据驱动决策**：通过提案分析和客户行为追踪，优化销售策略
- **业务流程整合**：与ERP系统无缝集成，实现业务闭环

### 1.4 商业模式
采用SaaS订阅模式，提供三个层级：
- **基础版（Basic）**：¥299/月/用户，适合小型工作室
- **专业版（Professional）**：¥599/月/用户，适合中型设计公司
- **企业版（Enterprise）**：¥999/月/用户，适合大型企业集团

## 2. 产品概述

### 2.1 产品定位
面向中国大陆市场的专业提案展示SaaS平台，专为设计公司、全屋定制公司和软装公司量身定制。

### 2.2 核心功能模块
1. **提案管理**：创建、编辑、分享、分析提案（支持12个模块类型）
2. **作品集管理**：展示案例作品，支持公共展示，从提案导入
3. **模板管理**：预设模板库，支持自定义模板
4. **预约管理**：客户预约、会议安排
5. **我的日历**：个人和团队日程管理
6. **AI集成**：智能图像生成、文案创作（贯穿所有12个模块）
7. **设计师管理**：团队成员管理
8. **系统设置**：个性化配置选项（包含品牌、AI、通知、订阅、ERP集成、账号权限）

### 2.3 技术架构
- **前端**：React + Next.js，响应式设计
- **后端**：Node.js + Express
- **数据库**：PostgreSQL + Redis
- **存储**：阿里云OSS（符合中国数据合规要求）
- **部署**：阿里云ECS，中国大陆境内服务器

## 3. 目标用户

### 3.1 用户画像

#### 3.1.1 设计公司
- **规模**：5-50人团队
- **痛点**：提案制作耗时、缺乏标准化、客户反馈难追踪
- **需求**：高效提案工具、品牌一致性、团队协作

#### 3.1.2 全屋定制公司
- **规模**：10-200人团队
- **痛点**：方案展示不够直观、报价复杂、客户决策周期长
- **需求**：3D效果展示、实时报价、客户互动

#### 3.1.3 软装公司
- **规模**：3-30人团队
- **痛点**：产品展示单一、搭配效果难呈现、供应链管理复杂
- **需求**：产品库管理、搭配方案、供应商对接

### 3.2 用户角色

#### 3.2.1 管理员（Admin）
- 拥有全部权限
- 可管理团队成员、设置系统参数、查看所有数据

#### 3.2.2 编辑者（Editor）
- 可创建、编辑、分享提案和作品集
- 可管理个人日历和预约
- 无权限管理其他用户

#### 3.2.3 查看者（Viewer）
- 仅可查看分配给自己的内容
- 无法进行编辑操作
- 可提供反馈和评论

## 4. 详细功能规格

### 4.1 提案管理模块

#### 4.1.1 提案创建
- 支持从模板快速创建新提案
- 三面板编辑界面（顶部导航 + 左侧章节导航 + 中央画布 + 右侧编辑面板）
- 实时自动保存（3-5秒防抖保存）和完整版本控制
- 支持多语言（中文、英文）

#### 4.1.2 提案编辑器布局（参考网站设计）

**三面板布局**：
- **顶部导航栏**（64px固定）：章节快速跳转、保存状态、预览、分享、用户菜单
- **左侧章节导航**（80px固定）：显示所有章节缩略图，点击跳转，支持拖拽排序
- **中央画布区域**（全屏滚动）：提案内容实时预览，所见即所得
- **右侧编辑面板**（450px可收起）：当前章节编辑控件、AI辅助功能

#### 4.1.3 提案模块系统（12个核心模块）

**基础模块**：
1. **Hero（英雄封面）**
   - 全屏背景图片
   - 项目标题（中英文）
   - 副标题（设计风格描述）
   - 客户名称
   - 进入按钮
   - AI支持：生成标题、副标题、背景图

2. **Insight（设计理念）**
   - 左右分栏布局（左文右图）
   - 章节标签（英文）+ 标题（中文）
   - 富文本内容编辑
   - 右侧图片支持灰度滤镜
   - 可调整左右比例（40/60、50/50、60/40）
   - AI支持：生成设计理念文案、生成配图

3. **Manifesto（设计宣言）**
   - 深色背景（#2D2D2D）
   - 左侧白色文字，右侧大尺寸图片
   - 布局比例：左30%右70%
   - AI支持：生成设计宣言、生成配图

**设计展示模块**：
4. **Floorplan（平面图对比）** ⭐
   - 前后对比功能：原始方案 vs 提案方案
   - 切换按钮（淡入淡出/左右滑动/上下滑动）
   - 对比说明文字
   - 默认显示设置

5. **Storage（收纳方案）** ⭐
   - 网格布局展示多个收纳区域
   - 每个区域包含：图片、区域名称、体积（m³）、功能说明
   - 支持2列或3列布局
   - AI支持：智能生成收纳方案

6. **Rendering（渲染图+热点标注）** ⭐
   - 主渲染图 + 热点标记系统
   - 热点坐标（x/y百分比）
   - 悬停显示产品详情（标签、描述、价格）
   - 右侧软装清单侧边栏
   - AI支持：生成渲染图、智能识别家具位置

7. **Gallery（效果图画廊）** ⭐
   - 分类展示（客厅、卧室、厨房、卫浴等）
   - 分类筛选标签
   - 网格布局（2/3/4列）或瀑布流
   - 图片点击放大
   - AI支持：批量生成效果图、智能分类

8. **Moodboard（情绪板）** ⭐
   - 材料色板（颜色块 + 色值 + 名称）
   - 参考图片不对称布局
   - 图片尺寸可调（小/中/大）
   - 图片透明度控制
   - AI支持：生成配色方案、推荐参考图片

**项目信息模块**：
9. **Technical（技术说明）**
   - 材料规格清单
   - 施工工艺说明
   - 环保标准
   - 单栏或双栏布局
   - AI支持：生成技术说明、生成材料清单

10. **Delivery（交付计划）**
    - 分阶段交付计划
    - 每阶段包含：名称、周期、内容清单
    - 总工期显示
    - AI支持：生成交付计划、优化工期安排

11. **Quotation（产品报价）**
    - 报价单编号和有效期
    - 产品清单表格（序号、名称、规格、数量、单价、小计）
    - 小计、税费、总计
    - 备注说明
    - 可选择已有报价单或创建新报价单
    - 支持显示产品图片和规格
    - AI支持：智能推荐产品、优化报价方案

12. **Ending（结尾页面）**
    - 感谢标题（中英文）
    - 公司信息（名称、电话、邮箱、地址）
    - 微信二维码、公众号二维码
    - 背景可选纯色或图片

#### 4.1.4 模块管理功能
- **添加模块**：点击左侧"+"按钮，从分类菜单选择模块
- **删除模块**：右键或编辑面板删除按钮
- **排序模块**：拖拽左侧章节导航调整顺序
- **复制模块**：快速复制现有模块
- **模块模板**：保存常用模块配置为模板

#### 4.1.5 自动保存和版本控制
- **自动保存**：3-5秒防抖保存，增量更新
- **保存状态指示**：顶部显示"已保存"/"保存中"/"未保存"
- **版本快照**：每次手动保存创建版本快照
- **版本历史**：查看所有历史版本列表
- **版本对比**：可视化对比两个版本的差异
- **版本回滚**：一键恢复到历史版本
- **版本元数据**：创建时间、作者、变更摘要

#### 4.1.6 协作编辑
- 多人同时编辑，实时同步
- 在线用户指示器
- 编辑冲突检测和解决
- 评论和批注功能

#### 4.1.7 提案分享
- **密码保护**：可设置访问密码
- **有效期控制**：可设置链接过期时间（1天-永久）
- **访问统计**：查看次数、停留时间、页面浏览热力图
- **分享渠道**：微信、邮件、短信、直接链接
- **水印保护**：可添加动态水印（访客IP、时间戳）

#### 4.1.8 提案分析
- 客户行为分析（哪些页面停留时间最长、是否完成查看）
- 转化率追踪（查看→咨询→成交）
- A/B测试功能（不同版本提案效果对比）

### 4.2 作品集管理模块

#### 4.2.1 作品展示
- 瀑布流布局，支持大图展示
- 最新作品优先，按年份分组显示
- 项目详情页（项目背景、设计理念、效果图展示）
- 多媒体支持（图片、视频、360°全景）
- 标签分类（风格、空间类型、预算范围等）

#### 4.2.2 从提案导入到作品集
- **用户自主选择章节**：勾选要保留的章节
- **推荐配置**：系统推荐保留视觉效果好的章节（Hero、Insight、Manifesto、Rendering、Gallery、Moodboard、Ending）
- **建议移除**：技术细节章节（Floorplan、Storage、Technical、Delivery、Quotation）
- **快速选择**：全选、推荐配置、仅图片、清空
- **实时预览**：显示将保留和移除的章节列表
- **作品信息编辑**：标题、简介、项目类型、完成时间
- **发布选项**：立即发布、设为精选作品

#### 4.2.3 作品集编辑
- 简化的编辑器（仅保留图片和文案模块）
- 不展示设计过程和报价信息
- 二次编辑导入的内容
- 优化视觉呈现效果

#### 4.2.4 公共展示
- **自定义域名**：支持绑定企业域名（如portfolio.company.com）
- **SEO优化**：自动生成SEO友好的URL、Meta标签、结构化数据
- **社交媒体分享**：一键分享到微信、微博、小红书等
- **移动端优化**：响应式设计，适配各种设备
- **年份分组**：按完成年份分组展示，最新在上
- **缩略图展示**：每个作品显示简介和缩略图
- **点击查看详情**：进入完整作品展示页面

#### 4.2.5 作品管理
- 批量上传和编辑
- 版权信息管理
- 访问权限控制（公开、私有、指定客户可见）
- 精选作品标记
- 作品排序（手动排序、按时间、按热度）

### 4.3 模板管理模块

#### 4.3.1 模板库
- 行业专属模板（住宅设计、商业空间、办公空间等）
- 风格模板（现代简约、新中式、欧式古典等）
- 场景模板（初次提案、深化方案、最终确认等）

#### 4.3.2 自定义模板
- 可基于现有提案创建模板
- 模板变量设置（客户名称、项目地址、预算等）
- 模板审批流程（管理员审核后发布）

#### 4.3.3 模板分发
- 团队模板共享
- 模板使用统计
- 模板版本管理

### 4.4 预约管理模块

#### 4.4.1 预约设置

- 预约类型设置（线上会议、现场勘测、方案讲解等）

#### 4.4.2 客户预约
- 客户自助预约界面
- 实时可用时间显示
- 预约信息确认（时间、地点、参与人员、准备材料）

#### 4.4.3 预约集成

- 自动生成会议议程和提醒

### 4.5 我的日历模块

#### 4.5.1 个人日历
- 日/周/月视图切换
- 事件创建和编辑（标题、时间、地点、描述、提醒）
- 重复事件设置（每日、每周、每月、每年）
- 事件颜色标记（不同类型用不同颜色区分）

#### 4.5.2 团队日历
- 团队成员日程可见性控制
- 共享日历创建
- 冲突检测（避免时间重叠）
- 忙碌状态显示

#### 4.5.3 日历集成
- 与主流日历应用同步（Outlook、Google Calendar、苹果日历）
- 微信日历提醒
- 企业微信/钉钉集成

#### 4.6.3 AI图像生成（MVP）
- **场景生成**：输入文字描述，自动生成室内场景图像
- **风格转换**：上传现有图片，转换为指定设计风格
- **材质替换**：在现有设计中替换材质（地板、墙面、家具等）
- **光照调整**：模拟不同时间段的光照效果
- **批量生成**：一次生成多张图片供选择
- **提示词优化**：AI自动优化用户输入的提示词

#### 4.6.4 AI文案创作（MVP）
- **提案文案**：根据项目信息自动生成专业提案文案
- **设计理念**：基于设计元素自动生成设计理念说明
- **产品描述**：为软装产品自动生成营销文案
- **多语言支持**：中英文自动翻译和优化
- **多版本生成**：一次生成3-5个版本供选择
- **文案风格**：可选择专业、亲和、奢华等不同风格

- **预算分配**：根据项目总预算，智能分配各环节预算
- **内容完整性检查**：检测提案是否缺少关键信息

#### 4.6.6 AI使用管理
- **使用统计**：显示本月AI使用次数和剩余额度
- **历史记录**：保存所有AI生成历史
- **收藏功能**：收藏优质生成结果
- **反馈机制**：对生成结果进行评分，优化模型
